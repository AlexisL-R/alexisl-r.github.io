Crochet hyperbolique

La référence de base pour le crochet hyperbolique est l'article de Henderson et Taimina dans The Mathematical Intelligencer, disponible https://link.springer.com/article/10.1007/BF03026623. Aussi disponible en ligne, en version préliminaire, sur le site web de Cornell http://pi.math.cornell.edu/~dwh/papers/crochet/crochet.html. Taimina a aussi fait un TEDx talk sur le sujet: https://www.youtube.com/watch?v=w1TBZhd-sN0.

Le patron employé est assez simple, à la base il faut faire un rang puis pour le prochain rang débuter par une double maille, continuer pour $N$ mailles et faire une double maille après et continuer aussi longtemps que voulu. La croissance est exponentielle alors vous devrez arrêter à un moment. Si cette petite explication n'est pas suffisante, allez voir ce site web avec un patron plus élaboré tiré de l'article de Taimina et Henderson https://theiff.org/press/interweave.html.
